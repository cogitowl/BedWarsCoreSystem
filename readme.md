# 部署

## 部署要求

1. Minecraft 版本 1.21.11 或更高版本

2. 服务器核心

- 原版（理论）
- Spigot（理论）
- Paper（测试可用且长期支持）

## 部署步骤

1. 下载数据包

   - 直接从 GitHub 仓库下载最新版本的 `BedWars.zip` 文件。

2. 安装数据包
    - 将下载的 `BedWars.zip` 文件解压缩。
    - 将解压后的 `BedWars` 文件夹放入服务器的 `datapacks` 目录中。

3. 启动服务器
    - 启动服务器，数据包将自动加载。

## 部署须知

- 本仓库并不包含任何 BedWars 地图，您需要自行下载或创建地图。
    - 推荐地图资源： [BedWars Maps](https://www.planetminecraft.com/projects/?keywords=BedWars) 
    - 地图放置基本要求
        - 所有的地图中心点z坐标必须为0，且地图必须以世界中心为原点进行构建。
        - 相隔地图中心点之间必须至少有1000格的距离，以避免干扰。
        - 例如：A地图在(1000, 0)，B地图在(2000, 0)，C地图在(3000, 0)。
        - 不要将地图放置在（0,0）以及负坐标区域，以避免潜在的兼容性问题。
    - 地图导入方法
        1. 下载或创建符合要求的地图。
        2. 放置在世界之中
        3. 通过命令 `/function debug`来打开调试面板，选择“新建地图”。
        4. 按照对应要求，导入地图即可。
- 数据包目前处于测试阶段，可能存在一些未发现的错误或不兼容性问题。请在使用过程中保持关注更新，并及时反馈任何问题。
